# Workshop Exercises

Each participant should create their own folder here to avoid merge conflicts.

## Folder Naming Convention

Use your first name or a unique identifier:
- `exercises/faiz/`
- `exercises/john/`
- `exercises/sarah/`

## What to Put in Your Folder

1. Create a simple Python file (e.g., `hello.py`)
2. Optionally add a `.gitignore` file
3. Optionally add a `README.md` with your notes
4. Any other files you want to practice with

## Example Structure

```
exercises/
├── faiz/
│   ├── hello.py
│   ├── .gitignore
│   └── README.md
├── john/
│   ├── hello.py
│   └── notes.txt
└── sarah/
    └── hello.py
```

## Why This Approach?

- **No conflicts:** Everyone works in separate folders
- **Easy to review:** Each PR only touches one folder
- **Real-world practice:** Similar to working on different features in a team

---

**Remember:** Never edit files in other participants' folders!
